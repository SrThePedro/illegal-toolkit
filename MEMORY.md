# IllegalKit Memory Bank

## Project Name
IllegalKit

## Tech Stack
- Python 3.10+
- yt-dlp
- rich
- questionary

## Goal
Provide a hacker-vibe, fast, and beautiful terminal UI for various tools.

## Rules
1. Always refer to this `MEMORY.md` file before adding new features.
2. Code must be in English.
3. UI must use `rich` for formatting/progress bars and `questionary` for interactive prompts.

## Packaging
- This project is an installable Python package via `pyproject.toml`
- The CLI entry point is the global `illegalkit` command
- Install with `pip install git+https://github.com/<username>/illegal-toolkit.git`
