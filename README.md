# IllegalKit

IllegalKit is an open-source Python CLI toolkit that replaces ad-filled websites for common internet tasks. It delivers a hacker-vibe, fast, and beautiful terminal UI built with `rich` and `questionary`.

## Tech Stack
- Python 3.10+
- `yt-dlp`
- `rich`
- `questionary`

## Features
- Terminal-first interface
- Interactive prompts
- Rich formatted output and progress bars

## Installation

### Install dependencies
```bash
pip install -r requirements.txt
```

### Install as a package
```bash
pip install .
```

### Install from GitHub
```bash
pip install git+https://github.com/<username>/IllegalKit.git
```

## Project Structure
- `MEMORY.md` - Project memory bank and rules
- `pyproject.toml` - Package metadata and entry point
- `illegalkit/` - Python package module
